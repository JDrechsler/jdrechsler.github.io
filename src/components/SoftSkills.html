• Strategy-Minded

    Throughout my extensive experience as a Full-Stack Software Engineer, I have worked with all the popular JS frameworks. I love JS but sometimes it is not necessary to use it. I am not a fan of using a framework just because it is popular. I like to use the right tool for the job.

  • Performance Oriented

    I love performance. I imagine creating apps like painting a picture. Without friction or refreshes or losing focus. Instant previews and no wait. The current tooling system does not allow for such a workflow which is why I am experimenting with creating my own framework | build | tool workflow that will allow me to create apps like painting a picture.

  • Accessibility Focused

    I did a 6-months Nanodegree with the Google Chrome engineers learning about the importance of accessibility and how to make websites accessible. I make sure that all my websites are accessible and user friendly.
